Project created by MERN stack
Topic of this project learn, improove and practice with graphql, express, node js, mongo db, react-apollo, apollo-boost

 
To start this project localy you need: 
1. Clone this repo
2. cd to this repo & npm i
3. Then cd to client & npm i
4. Then cd .. & npm run dev

UI started on port 3000
Backend started on port 3005
